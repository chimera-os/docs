gnome-maps-42.3

Introduction to GNOME Maps

GNOME Maps is a map application for GNOME.

This package is known to build and work properly using an LFS-11.2 platform.

Package Information

  • Download (HTTP): https://download.gnome.org/sources/gnome-maps/42/gnome-maps-42.3.tar.xz

  • Download (FTP): ftp://ftp.acc.umu.se/pub/gnome/sources/gnome-maps/42/gnome-maps-42.3.tar.xz

  • Download MD5 sum: f7842a970a6d64dec0cc68a9da77ba5c

  • Download size: 2.2 MB

  • Estimated disk space required: 19 MB

  • Estimated build time: less than 0.1 SBU

GNOME Maps Dependencies

Required

clutter-gtk-1.8.4, desktop-file-utils-0.26, Folks-0.15.5, GeoClue-2.6.0, geocode-glib-3.26.4, gfbgraph-0.2.5, Gjs-1.72.2, libchamplain-0.12.20, libgweather-4.0.0, libgee-0.20.5, libhandy-1.6.3, and rest-0.8.1

User Notes: https://wiki.linuxfromscratch.org/blfs/wiki/gnome-maps

Installation of GNOME Maps

Install GNOME Maps by running the following commands: 

mkdir build &&
cd    build &&

meson --prefix=/usr --buildtype=release .. &&
ninja

This package does not come with a test suite.

Now, as the root user: 

ninja install

Contents

Installed Programs: gnome-maps (symlink)
Installed Libraries: libgnome-maps.so
Installed Directories: /usr/lib/gnome-maps and /usr/share/gnome-maps
