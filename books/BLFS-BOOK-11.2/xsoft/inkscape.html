Inkscape-1.2.1

Introduction to Inkscape

Inkscape is a what you see is what you get Scalable Vector Graphics editor. It is useful for creating, viewing and changing SVG images.

This package is known to build and work properly using an LFS-11.2 platform.

Package Information

  • Download (HTTP): https://inkscape.org/gallery/item/34673/inkscape-1.2.1.tar.xz

  • Download MD5 sum: 9e0acbb3dfe8c62953c788134d208dc2

  • Download size: 94 MB

  • Estimated disk space required: 1.0 GB (182 MB installed)

  • Estimated build time: 6.5 SBU (with parallelism=4)

Note

The tarball inkscape-1.2.1.tar.xz will extract to the directory inkscape-1.2.1_2022-07-14_dc2aedaf03/.

Inkscape Dependencies

Required

Boost-1.80.0, double-conversion-3.2.1, GC-8.0.6, GDL-3.40.0, Gsl-2.7.1, Gtkmm-3.24.6, libsoup-2.74.2, libxslt-1.1.36, Poppler-22.08.0, popt-1.18 and Wget-1.21.3 (to download the test dependencies)

Recommended

ImageMagick-7.1.0-46, Little CMS-2.13.1 or Little CMS-1.19, libcanberra-0.30 (to eliminate plugin warnings), Potrace-1.16 (for the bucket-fill tool), lxml-4.9.1 and Scour-0.38.2 (both at runtime, for Save As Optimized SVG).

Optional

Aspell-0.60.8, dbus-1.14.0 (to run inkscape from scripts), Doxygen-1.9.4, gspell-1.10.0, libcdr, libvisio, libwpg (or libwpd) and NumPy (at runtime for some extensions)

Optional Runtime Dependencies (for some of the Inkscape extensions)

XML::XQL

User Notes: https://wiki.linuxfromscratch.org/blfs/wiki/inkscape

Installation of Inkscape

Install Inkscape by running the following commands: 

mkdir build                       &&
cd    build                       &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      ..                          &&
make

This package does not come with a working test suite.

Now, as the root user: 

make install

Note

This package installs icon files into the /usr/share/icons/hicolor hierarchy and desktop files into the /usr/share/applications hierarchy. You can improve system performance and memory usage by updating /usr/share/icons/hicolor/index.theme and /usr/share/applications/mimeinfo.cache. To perform the update you must have GTK+-2.24.33 or GTK+-3.24.34 installed (for the icon cache) and desktop-file-utils-0.26 (for the desktop cache) and issue the following commands as the root user: 

gtk-update-icon-cache -qtf /usr/share/icons/hicolor &&
update-desktop-database -q

Command Explanations

-DCMAKE_BUILD_TYPE=Release: This switch is used to build the release library without any debug `assert` in the code.

-DWITH_DBUS=ON: use this if you wish to use inkscape in interactive scripts which manipulate images.

Contents

Installed Programs: inkscape and inkview
Installed Library: libinkscape_base.so
Installed Directories: /usr/lib/inkscape and /usr/share/inkscape

Short Descriptions

inkscape

is a SVG (Scalable Vector Graphics) editing program

inkview

is a simple program for displaying SVG files

libinkscape_base.so

provides the routines used by inkscape and inkview
